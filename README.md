🎯 Hangman Game

A classic word guessing game built using Python where the player tries to guess the hidden word letter by letter.

📌 Overview

This project implements the traditional Hangman game in a command-line interface. The player guesses letters, and the program reveals correct guesses while limiting the number of incorrect attempts.

🚀 Features

✔️ Random word selection from a predefined list
✔️ Interactive user input for guessing letters
✔️ Displays progress with hidden and revealed letters
✔️ Tracks incorrect guesses with a limit
✔️ Win/Lose condition handling

🛠️ Tech Stack
Language: Python
Concepts Used:
Functions
Loops
Conditional statements
Lists and strings
Random module
⚙️ How to Run
python hangman_game.py
🧠 Game Logic
The system selects a random word
User guesses one letter at a time
Correct guesses reveal the word
Incorrect guesses reduce chances
Maximum 6 wrong attempts allowed
📊 Example Gameplay
Welcome to Hangman!

Word to guess: _ _ _ _ _ _

Word to guess: _ a _ _ _ _
